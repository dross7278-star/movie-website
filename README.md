# Action Movies Landing Page

A modern, responsive landing page for an action movie themed website.

## Features

- Hero section with call-to-action buttons
- Feature cards and highlights section
- Responsive layout for desktop and mobile
- GitHub Pages workflow included
- VS Code workspace autosave settings included

## Project files

- `index.html` - landing page markup
- `style.css` - visual design and responsive styles
- `javascript.js` - movie search behavior and UI logic
- `.github/workflows/deploy-pages.yml` - GitHub Pages deployment workflow
- `.vscode/settings.json` - workspace autosave settings

## Local preview

Open `index.html` in your browser.

## Deploy to GitHub Pages

1. Push this repository to GitHub on branch `main`.
2. In GitHub, open the repository and go to `Actions`.
3. Wait for the workflow `Deploy static site to GitHub Pages` to complete.
4. Open your published Pages URL from the deployment output.
