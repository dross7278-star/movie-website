# Action Movies Landing Page

A modern, responsive landing page for an action movie themed website.

## Features

- Hero section with call-to-action buttons
- Feature cards and highlights section
- Responsive layout for desktop and mobile
- GitHub Pages compatible (branch deployment)
- VS Code workspace autosave settings included

## Project files

- `index.html` - landing page markup
- `style.css` - visual design and responsive styles
- `javascript.js` - movie search behavior and UI logic
- `.vscode/settings.json` - workspace autosave settings

## Local preview

Open `index.html` in your browser.

## Deploy to GitHub Pages

1. Push this repository to GitHub on branch `main`.
2. In GitHub, open the repository and go to `Settings` > `Pages`.
3. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
4. Set `Branch` to `main` and folder to `/ (root)`, then save.
5. Wait 1-2 minutes, then open `https://dross7278-star.github.io/movie-website/`.
